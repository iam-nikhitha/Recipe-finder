# Recipe Finder

## Project Description
Recipe Finder is a web application that allows users to search, save, and share recipes. The app includes features such as user authentication, search functionality via an API, and a modern user interface.

## Features
- User Registration and Login
- Search for recipes using API
- Save favorite recipes
- Share recipes with others
- Dark/Light mode for enhanced UI
